# 基于Python+Django的图书管理系统 python109

## 项目概述

本项目是一个基于Python语言和Django框架开发的图书管理系统。该系统旨在通过简洁的界面和实用的功能，帮助用户高效管理图书资料，提供便捷的借阅服务。

## 技术栈

- **后端**: Python 3.x, Django
- **数据库**: SQLite
- **文档**: Markdown

## 功能模块

- **登录模块**: 用户登录与权限验证
- **图书管理**: 对图书进行增删改查操作
- **借阅管理**: 管理借阅信息，记录借还日期
- **个人管理**: 用户可查看和编辑个人信息
- **会员管理**: 管理会员信息，提供会员服务
- **数据可视化**: 展示图书借阅数据的图表分析

## 系统角色

- **管理员**: 管理系统用户和图书信息
- **普通用户**: 进行图书借阅和个人信息管理

## 运行环境

- 操作系统: 兼容Windows、Linux和MacOS
- Python版本: Python 3.x
- Django版本: 需与Python版本兼容的最新版本
- 数据库: SQLite

## 部署步骤

1. 确保Python和Django环境已正确安装
2. 克隆或下载项目源码
3. 导入数据库文件（SQLite）
4. 更新配置文件，包括数据库配置
5. 运行Django的迁移命令来创建或更新数据库表
6. 启动开发服务器进行测试
